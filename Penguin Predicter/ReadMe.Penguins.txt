# Penguin Species Classifier #


### Overview
This project uses a Decision Tree Classifier to predict species of penguins based on several features such as flipper length, culmen depth, and others.

### Dataset
From Kaggle: palmer-archipelago-antarctica-penguin-data/penguins_size.csv
You can download from here: https://www.kaggle.com/code/parulpandey/penguin-dataset-the-new-iris/notebook

###How to Use
Replace penguins_lter.csv with your dataset.
Run the code to train the model.
Save the trained model using joblib.
Load the model and make predictions as needed.
